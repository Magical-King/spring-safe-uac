# spring-safe-uac

统一权限管理业务快
===========================

###########环境依赖
jdk1.8
redis ~

###########部署步骤

1. mvn install // 打包至本地maven

2. 业务模块直接引入父级依赖

3. SPI自动加载即可，业务子模块配置默认配置项直接启动

###########功能介绍

1. 角色权限控制

2. 国际化

3. 邮箱通知、告警
